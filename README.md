# Project ShiraCo. 

一个基于 `python-telegram-bot` 开发的轻量级 Telegram 机器人。

[![License: AGPL v3](https://img.shields.io)](https://www.gnu.org)
[![Python 3.10+](https://img.shields.io)](https://www.python.org)

## 🌟 功能特性

- 📩 **私信转发**：自动将用户发给机器人的私信转发给管理员，支持管理员直接回复。
- 🖼️ **一键涩图**：集成 Lolicon API，支持通过标签（Tag）和分级（R18/混合）搜索图片。
- 🔍 **账号查询**：
    - `/id`：快速获取用户的 Telegram ID。
    - `/dc`：通过头像解析用户所在的 Telegram 数据中心（DC）。
